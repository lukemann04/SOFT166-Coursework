Instructions

On the Game page you will find a game that uses colour sequences to test your memory.


There is a dropdown list which can be used to select a specific light for the game. If a light is not selected, light 1 will be used.


1. To start the game press the "Start Sequence" button. This will display the first sequence.

2. Try to memorise the colours shown in the sequence. When the sequence is over, press the coloured buttons in the same order as the sequence displayed.

3. If your answer is correct, the background will turn green. If your answer is incorrect, the background will turn red.

4. If your answer is correct, when you next press the show sequence button, the sequence will be one colour longer.

5. If your answer is incorrect, the length of the next sequence will return to 1.

6. Do your best to memorise the colours and see how high you can score!
