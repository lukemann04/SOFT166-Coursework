About Page

The game on this website makes use of the Hue lights in SMB109. It's recommended that you play the game on a computer in this room for the best experience.


However, the game is still playable if you are unable to do so.

More Info

The website is able to connect to the Hue lights in SMB109 by using JQuery. The lights are controlled by a router called a bridge.


The website is able to send HTTP requests to the IP address of the bridge in order to manipulate the lights.
